# Kotlin Spring Boot App For Demo

## Add Open Api Generate Code Dependency

    dependecnies {
        ...
        implementation(files("$projectDir/../kotlin-spring-boot-specification2/target/openapi-spring-1.0.0.jar"))
        ...
    }

## Run your project

    ./gradlew bootRun
